Step 1 — Build the Android APK

  Run this in PowerShell or Git Bash:
  cd "D:\My Projects\Unix\android"
  ./gradlew assembleRelease

  Output APK will be at:
  D:\My Projects\Unix\android\app\build\outputs\apk\release\app-release.apk

  ---
  Step 2 — Upload APK to server

  # Create downloads folder on server
  ssh -i "D:\My Projects\Deployment\Keys" root@204.168.146.103 "mkdir -p /opt/centrixpro/downloads"

  # Upload the APK
  scp -i "D:\My Projects\Deployment\Keys" \
    "D:\My Projects\Unix\android\app\build\outputs\apk\release\app-release.apk" \
    root@204.168.146.103:/opt/centrixpro/downloads/unix.apk
